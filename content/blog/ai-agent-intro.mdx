## 什么是 AI Agent？

AI Agent 是一个能**自主感知环境、做出决策并执行行动**的智能系统。与单纯的 LLM 调用不同，Agent 具备：

- **工具使用**（Tool Use）：调用外部 API、数据库、搜索引擎
- **记忆系统**（Memory）：短期记忆（上下文窗口） + 长期记忆（向量数据库/RAG）
- **规划能力**（Planning）：分解复杂任务、多步骤执行
- **反馈循环**（Feedback Loop）：根据执行结果自我修正

## 主流框架

| 框架 | 特点 | 适合场景 |
|------|------|---------|
| LangChain | 生态最丰富 | 快速原型 |
| CrewAI | 多 Agent 协作 | 团队任务 |
| AutoGPT | 自主规划 | 复杂长任务 |
| MCP | 标准化工具协议 | 工具集成 |

## MCP 协议

MCP (Model Context Protocol) 是 Anthropic 提出的标准化协议，让 LLM 与外部工具交互：

```python
# MCP 工具定义示例
@mcp.tool()
async def search_docs(query: str) -> str:
    """搜索文档库"""
    results = await vector_db.similarity_search(query)
    return format_results(results)
```

## 关键设计模式

1. **ReAct** (Reasoning + Acting)：思考→行动→观察→再思考的循环
2. **Plan-and-Execute**：先规划再执行
3. **Multi-Agent**：多个专业 Agent 协作
