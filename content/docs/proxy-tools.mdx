[ArgoV](https://github.com/m2dumpling/ArgoV) 是双内核零信任代理面板，单 Bash 脚本一键部署。支持 VLESS / VMess / Reality / Hysteria2 / TUIC / AnyTLS / Shadowsocks 七种协议，内置多用户配额、WARP 分流、聚合订阅。

[项目详情 →](/projects/proxy-tools)

## 快速安装

```bash
bash <(curl -Ls https://raw.githubusercontent.com/m2dumpling/ArgoV/main/argov.sh)
```

安装后输入 `ag` 打开面板。
